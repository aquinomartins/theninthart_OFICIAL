# Imagens da experiência Tempo

Os arquivos binários de imagem não são versionados neste pacote porque não são compatíveis com o fluxo atual de revisão.

Mantenha estes nomes e caminhos ao inserir os assets finais pelo GitHub:

- `tempo/images/imagem013.png`
- `tempo/images/imagem010.png`
- `tempo/images/imagem1.png`
- `tempo/images/imagem2.png`
- `tempo/images/imagem014.png`
- `tempo/images/imagem4.png`

`tempo/index.html` já aponta para esses caminhos relativos. Quando os arquivos forem adicionados com esses nomes, a página não exigirá mudanças adicionais de HTML, CSS ou JavaScript.
